## Time Complexity Order

| Name   |      Notation      | 
|----------|:-------------|
| Constant | O(1)  |
| Logarithmic |  O(log n)     |
| Linear | O(n) |
| N-LogN | (n log n) |
| Quadratic | O(n^2) |
| Polynomial | O(n^c) c constant > 1 |
| Exponential | O(c^m) c constant > 1 |
| Factorial | O(n!) / O(n^n) |
