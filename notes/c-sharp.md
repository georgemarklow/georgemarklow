### Strings

#### Compare
```csharp
string.Compare("a", "a")
string.Compare("a", "b")
string.Compare("b", "a");
string.Compare("A", "a");
```
#### CompareTo
```csharp
var s1 = "a";               
s1.CompareTo(s1);
s1.CompareTo("a");
```
#### Concat
```csharp
string.Concat("a", "b")
```
#### Contains
```csharp
"abc".Contains("A")
"abc".Contains("a")
```
